📝 Objetivo del Ejercicio

En este ejercicio aprenderemos a conectarnos a una base de datos MySQL utilizando PHP. Es fundamental entender cómo establecer una conexión segura y eficiente.

💡 Nota Importante:

Asegúrate de tener instalado XAMPP o WAMP antes de comenzar.

🔍 Desarrollo

Paso 1: Configuración de la base de datos

Primero, necesitamos crear nuestra base de datos. Accede a phpMyAdmin y crea una nueva base de datos llamada "mi_proyecto".

Paso 2: Código de conexión

Ahora vamos a escribir el código PHP para conectarnos a nuestra base de datos: 

<?php
// Datos de conexión
$servidor = "localhost";
$usuario = "root";
$password = "";
$baseDatos = "mi_proyecto";

// Crear conexión
$conexion = new mysqli($servidor, $usuario, $password, $baseDatos);

// Verificar conexión
if ($conexion->connect_error) {
    die("Error de conexión: " . $conexion->connect_error);
}

echo "Conexión exitosa!";
?>
⚠️ ¡Atención!

Nunca subas tus credenciales de base de datos a un repositorio público.

Paso 3: Probar la conexión

Para probar que todo funciona correctamente, guarda el archivo como conexion.php y ábrelo en tu navegador desde localhost.

✨ Conclusiones

En este ejercicio hemos aprendido:

  • Cómo crear una base de datos en phpMyAdmin
  • La sintaxis básica para conectarse con mysqli
  • Cómo verificar si la conexión fue exitosa
  • Buenas prácticas de seguridad básicas
